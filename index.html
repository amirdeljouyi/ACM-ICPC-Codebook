Dynamic Programming

Unbounded Knapsack

Returns max value that you can fit inside of a given space. Assume not tested, from old codebook. Variables int N: Number of items int W: Max weight int[] v: Item values int[] w: Item weights

public static int knapsack(int N, int W, int[] v, int[] w) {
    int[] m = new int[W*N];

    for (int j = 0; j < W + 1; j++) {
        for (int i = 0; i < N; i++) {
            if (w[i] <= j) {
                m[j] = max(m[j], m[j-w[i]] + v[i]);
            }
        }
    }

    return m[W + (N-1)];
}

0-1 Knapsack

Only psuedocode right now :( Define m[i,w] to be the maximum value that can be attained with weight less than or equal to w using items up to i (first i items). The solution can then be found by calculating m[n,W]. To do this efficiently we can use a table to store previous computations.

// Input:
// Values (stored in array v)
// Weights (stored in array w)
// Number of distinct items (n)
// Knapsack capacity (W)
for j from 0 to W do
  m[0, j] := 0
end for 
for i from 1 to n do
  for j from 0 to W do
    if w[i] <= j then
      m[i, j] := max(m[i-1, j], m[i-1, j-w[i]] + v[i])
    else
      m[i, j] := m[i-1, j]
    end if
  end for
end for

Basic structure of a solution

"The only way to call a non-static method from a static method is to have an instance of the class containing the non-static method." - StackOverflow.com In order to use instances of our own classes this is how we must structure our boilerplate code. It's only about 3 more lines of typing, but worth it for the hassle it can save.

import java.util.*;

class Main {

    // Spawns an instance of itself so we can use non-static objects
    public static void main(String[] args) {
        Main m = new Main();
        m.begin();
    }

    // True entry point
    public void begin() {
        Scanner in = new Scanner(System.in);

        System.out.println("I am a non static object");    
        // etc...
    }
}

Greatest Common Denominator

Implemented using Euclids algorithm.

public static long gcd(long a, long b) {
    return (b == 0) ? a : gcd(b, a % b);
